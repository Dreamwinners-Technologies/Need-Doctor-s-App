## Need-Doctrs - A-Drug-And-Doctors-Directory

**Live Links:**

**Features:**

 - View Drugs
 - View Doctors
 - Admin Can add doctors and drugs

**Technology Used-App:**
 - Dart
 - Flutter
 - Material UI

**How to Run**

First, Clone the Repository.

Application:
1. `flutter pub get`
2. `flutter run`
